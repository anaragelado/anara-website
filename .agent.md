# AI Coding Agent Master Directives

You are an expert web developer building a high-end, mobile-first Single Page Application (SPA) for "Anara Gelado Artesanal", an artisanal ice cream brand.

## 1. Project Workflow
Your primary source of truth for tasks is the `roadmap.md` file in the root directory. 
* Always read `roadmap.md` before starting a task to understand current progress and objectives.
* Update `roadmap.md` upon completing major milestones if instructed to do so.

## 2. Context Files (Required Reading)
Before making architectural, design, or logic decisions, you must read the relevant context files located in the `.agent/context/` folder:

* `01-tech-stack.md`: Core frameworks, libraries, and coding standards.
* `02-brand-guidelines.md`: Colors, typography, and brand vibe.
* `03-architecture-content.md`: SPA structure, sections, and data flow.
* `04-ui-layout-specs.md`: Mobile-first rules, rounded corners, and spacing constraints.

## 3. Web Design Skills
* Strictly follow the instructions and best practices defined in `.agent/skills/skill-webdesign/SKILL.md`.

## 4. Asset Management
* Base assets (logo, favicon): Located in the root `assets/` directory.
* Media files (images, videos): Located in `public/assets/images/` and `public/assets/videos/`. Use relative paths pointing to the public folder for rendering.

**Core Rule:** Do not guess the brand colors, layout requirements, or tech stack. Always pull parameters from the `.agent/context/` files.
